📦 Print Bell Pro - Aplikasi Cetak Label Resi Profesional
 
Aplikasi web untuk mencetak label resi yang dapat disesuaikan, mendukung resi dari marketplace dan resi manual. Cocok untuk penjual online yang ingin mengoptimalkan proses pencetakan resi dengan tampilan profesional dan fitur lengkap.
 
✨ Fitur Utama
 
🆓 Versi Gratis
 
- Cetak resi dari marketplace (dukungan file CSV)
- Buat resi manual dengan data penerima/pengirim
- Pilihan ukuran kertas thermal: 58mm/80mm/100mm
- Tambahkan catatan tambahan (misal "Wajib Video Unboxing")
- Preview label sebelum dicetak
- Integrasi langsung ke printer via browser
- Batasan: Maksimal 10 resi per hari
 
💎 Versi Pro (Berbayar)
 
Semua fitur gratis +:
 
- Cetak resi masal dalam jumlah berapapun
- Dukungan file CSV & Excel (.xlsx)
- Crop otomatis elemen resi marketplace (sembunyikan logo MP/detail kurir)
- Upload & simpan logo toko secara permanen
- Simpan template label favorit
- Tidak ada batasan jumlah cetak per hari
- Tidak ada watermark
 
🚀 Cara Menggunakan
 
1. Akses Aplikasi
 
Buka link aplikasi di browser HP/komputer:
 https://namausernamamu.github.io/print-label-resi-app/ 
 
2. Resi Marketplace
 
- Unggah File: Siapkan file CSV dengan kolom:  pengirim ,  penerima ,  hp_penerima ,  alamat ,  no_resi ,  marketplace 
- Pilih Pesanan: Pilih pesanan yang ingin dicetak dari daftar
- Sesuaikan: Pilih ukuran kertas dan tambahkan catatan
- Cetak: Klik tombol "Cetak Label" atau simpan sebagai PDF
 
3. Resi Manual
 
- Isi Data: Masukkan nama toko/pengirim, data penerima, alamat, dan nomor resi
- Sesuaikan: Pilih ukuran kertas dan tambahkan catatan
- Preview & Cetak: Lihat pratinjau lalu cetak
 
4. Pengaturan
 
- Ubah tema aplikasi (Terang/Gelap)
- Simpan logo toko secara permanen (hanya Pro)
- Kelola pengaturan lain sesuai kebutuhan
 
🖨️ Integrasi Printer
 
Aplikasi mendukung pencetakan langsung ke printer thermal melalui:
 
- Fitur print bawaan browser (kompatibel dengan semua printer)
- Web Bluetooth/Web USB API (untuk koneksi langsung tanpa kabel - perlu dukungan printer)
 
💻 Teknologi yang Digunakan
 
- HTML5 & CSS3 (Desain responsif dan profesional)
- JavaScript (Fungsi utama aplikasi)
- Library Eksternal:
- JSBarcode (Generate barcode)
- QRCode.js (Generate QR Code)
- PapaParse (Baca file CSV)
- jsPDF (Simpan label sebagai PDF)
- Hosting: GitHub Pages (gratis dan mudah diakses)
