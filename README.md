# 🚀 ResiPro Premium - Solusi Cetak Label Thermal HP

**ResiPro** adalah tool berbasis web gratis yang dirancang khusus untuk seller Shopee, TikTok Shop, dan Tokopedia yang ingin cetak resi menggunakan printer thermal (100x150mm) langsung dari HP tanpa ribet.

## ✨ Fitur Utama
- **Auto-Crop:** Memotong resi A4 menjadi ukuran label thermal 10x15cm secara otomatis.
- **Bulk Merge:** Gabungkan banyak file PDF resi menjadi satu file panjang (siap print sekaligus).
- **Custom Branding:** Masukkan **Logo Toko** dan **Nama Toko** langsung di dalam label pengiriman.
- **Security Watermark:** Otomatis menambahkan teks *"WAJIB VIDEO UNBOXING"* untuk keamanan seller.
- **100% Aman & Privat:** Proses pengolahan PDF dilakukan di browser HP Anda (Client-side). Data tidak dikirim ke server mana pun.

## 🛠️ Cara Penggunaan
1. Download label pengiriman (PDF) dari aplikasi Marketplace.
2. Buka [Link Website Kamu Di Sini]
3. Masukkan Nama Toko dan Upload Logo (Opsional).
4. Pilih satu atau banyak file PDF resi yang sudah didownload.
5. Klik **"GABUNG & CETAK RESI"**.
6. File siap print akan otomatis terdownload ke folder Download HP Anda.

## 📱 Cocok Untuk:
- Printer Thermal Bluetooth/USB (Ukuran 100mm x 150mm).
- Seller yang hanya menggunakan HP (tanpa PC/Laptop).
- Semua Marketplace (Shopee, TikTok, Tokopedia, Lazada, dll).
